# AI Tool Pricing Data

This document contains the verified pricing data for various AI tools, which serves as the basis for our hardcoded audit rules engine.

## Code Editors
### Cursor (https://cursor.sh/pricing)
- **Hobby:** Free ($0/mo) - Basic code completions, limited premium model requests.
- **Pro:** $20/user/mo - 500 fast premium requests, unlimited slow premium requests, unlimited completions.
- **Team:** $40/user/mo (annual: $40/user/mo) - Everything in Pro + centralized billing, admin dashboard, team policies.
- **Enterprise:** Custom - SSO, SOC2 compliance, no telemetry.

### GitHub Copilot (https://github.com/features/copilot/pricing)
- **Free:** $0/mo - Basic autocomplete, limited chat, standard models.
- **Pro:** $10/user/mo - Individual developers. Unlimited chat, autocomplete.
- **Business:** $19/user/mo - Organizations. Policy management, IP indemnity.
- **Enterprise:** $39/user/mo - Custom fine-tuning, enterprise search.

### Windsurf (Codeium) (https://codeium.com/windsurf/pricing)
- **Free:** $0/mo - Basic completion, limited chat.
- **Pro:** $15/user/mo - Unlimited fast premium usage, more contexts.
- **Team:** $40/user/mo - Centralized billing, team analytics.
- **Enterprise:** Custom / $150/user/mo approx - Self-hosted, custom training, strict privacy.

## Conversational AI / General LLMs
### Anthropic Claude (https://www.anthropic.com/pricing)
- **Free:** $0/mo - Basic access to Claude 3.5 Sonnet.
- **Pro:** $20/user/mo - 5x usage, early access to new features.
- **Team:** $25/user/mo (annual), $30/user/mo (monthly) - Centralized billing, higher usage limits.
- **Enterprise:** Custom - Admin controls, SCIM, Audit logs.

### OpenAI ChatGPT (https://openai.com/chatgpt/pricing)
- **Free:** $0/mo - Basic access.
- **Plus:** $20/user/mo - Access to GPT-4, DALL-E, data analysis.
- **Team:** $25/user/mo (annual), $30/user/mo (monthly) - Higher limits, collaborative workspace.
- **Enterprise:** Custom - Unlimited high-speed GPT-4, SSO, enterprise data privacy.

### Google Gemini (https://gemini.google.com/advanced)
- **Free:** $0/mo - Basic Gemini.
- **Advanced (Google One AI Premium):** $20/user/mo - Access to Gemini 1.5 Pro, integration in Workspace.
- **Enterprise / Workspace:** Variable per Google Workspace plan add-ons (typically $20-30/user/mo).
